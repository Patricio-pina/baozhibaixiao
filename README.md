# 包治百消 - 微信小游戏版

一款以名牌包包为主题的消消乐微信小游戏，拥有收集名牌包图鉴的长期发展目标，优雅、浪漫、魔性的音效，专为女性用户设计。

## 游戏特色

- **名牌包包主题**：游戏中包含多种知名奢侈品牌的经典包款，如Luminelle月光诗语、Aetheris星轨之约、Sylvanscape森语秘境、Chronosette时光信使、Nimbelle云端织梦等
- **包包图鉴收集**：通过完成关卡解锁新包包，收集完整的奢华包包图鉴
- **精美视觉设计**：优雅浪漫的界面风格，精致的包包图案
- **魔性音效系统**：匹配成功、解锁新包包等操作都有独特的音效反馈
- **关卡进阶系统**：30个精心设计的关卡，难度逐渐提升，挑战性增强
- **特殊目标系统**：高级关卡拥有特殊目标，增加游戏的策略性
- **社交排行榜**：与好友比拼，查看谁收集的包包最多，分数最高
- **分享功能**：将游戏分享给好友，一起享受包治百消的乐趣

## 如何运行

1. 下载并安装[微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)
2. 克隆或下载本仓库
3. 在微信开发者工具中导入项目，选择项目根目录
4. 在"详情"面板中，勾选"不校验合法域名"和"启用云开发"
5. 点击"编译"按钮，预览游戏

## 游戏玩法

- 点击并交换相邻的两个包包，形成三个或更多相同包包的横向或纵向连线
- 连线的包包会被消除，并获得相应分数
- 连续消除可获得连击加分
- 达到目标分数即可通关，解锁新关卡和新包包

## 技术栈

- JavaScript (ES6+)
- 微信小游戏API
- 微信云开发
- Canvas 2D渲染

## 云开发配置

1. 在微信开发者工具中，点击"云开发"按钮
2. 创建云开发环境（如果没有）
3. 复制云环境ID，替换`game.js`中的`env`参数
4. 创建以下数据集合：
   - `user_collection`: 用户收集的包包数据
   - `user_progress`: 用户游戏进度数据
   - `leaderboard`: 排行榜数据

## 注意事项

- 游戏主包大小控制在8MB以内，符合微信小游戏要求
- 图片资源需要压缩优化，减少加载时间
- 音频资源采用低比特率MP3格式，减小体积
- 游戏适配不同尺寸的手机屏幕
- 游戏支持横屏和竖屏模式
- 游戏支持触摸和鼠标操作
